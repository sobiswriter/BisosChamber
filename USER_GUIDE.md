# 📖 User Guide - Biso's Chamber

Welcome to **Biso's Chamber**, your cozy sanctuary for AI conversations.

---

## 🎯 Getting Started

### First Time Setup

1. **Get Your API Key** 
   - Visit [Google AI Studio](https://aistudio.google.com/app/apikey)
   - Click "Create API Key"
   - Copy your key

2. **Enter API Key in the App**
   - Click the ⚙️ Settings button (bottom of sidebar)
   - Paste your API key
   - Click "Save Settings"

3. **Create Your First Persona**
   - Click "MANIFEST" button in the sidebar
   - Fill in the persona details:
     - **Name**: What should they be called?
     - **Role**: Their archetype (e.g., "Grumpy Historian")
     - **Bio**: Their backstory
     - **Personality**: Their vibe
     - **Greeting**: How they say hello
   - Choose an avatar and theme color
   - Click "Manifest Soul"

---

## 💬 Using the Chamber (Chat Mode)

### Starting a Conversation
1. Select a persona from the sidebar
2. Type your message in the input bar
3. Press Enter or click Send (➤)

### Advanced Features
- **📎 Attach Media**: Click the paperclip to share images, videos, or audio
- **🎭 Scenario Mode**: Click the theater icon to set up a scene with an image
- **➕ New Chat**: Start fresh while keeping history
- **🕒 Memory Lane**: View past conversations
- **📤 Export**: Save your chat as JSON
- **📥 Import**: Load a saved chat

### Keyboard Shortcuts
- `Alt + A`: Focus chat input
- `Alt + S`: Focus persona list
- `Alt + N`: Create new persona
- `Alt + C`: Open Chamber view
- `Alt + P`: Open Theater view

---

## 🎭 Using the Theater (Playground Mode)

### Setting Up a Scene

1. **Switch to Theater**
   - Click "THEATER" tab in the sidebar
   - Click "NEW SESSION"

2. **Cast Your Characters**
   - Select 2-4 personas
   - Set their "scene roles" (e.g., "The Accused", "The Detective")
   - Arrange turn order by dragging

3. **Define the Scene**
   - Write the premise/setting
   - Click "Raise the Curtain"

### During the Play
- **Next Soul**: Manual turn-by-turn progression
- **Auto-Play**: Let them talk continuously
- **Cast Shift**: Inject dramatic events

### Edit Your Profile

2. Set your name, bio, and personality
3. Choose an avatar

## 🎨 Customization

### Themes

## 🐛 Troubleshooting

### "API Key Required" Error
- Go to Settings
- Enter a valid Gemini API key
- Make sure you have quota remaining

### Blank Screen After Clear Chat
- Click "New Conversation"
- Or select a different persona
- Or refresh the page

### Images Not Loading
- Check file size (< 5MB recommended)
- Supported formats: JPG, PNG, GIF, WebP
- Images are auto-compressed

### Mobile Sidebar Won't Close
- Tap the darkened area outside the sidebar
- Or tap the ☰ hamburger menu again
